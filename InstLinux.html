Configuration d'un serveur web

Présentation du projet

Lors du second semestre de la première année de BUT, nous avons du réaliser l'installation d'un serveur web. Ce projet devait être réalisé de manière individuelle. Dans un premier temps, nous avons du installer Debian sur une machine virtuelle. Pour ce projet, il nous était demandé d'installer une machine virtuelle qui soit uniquement textuelle. Une fois Debian configuré et la machine virtuelle opérationnelle, nous devions installer divers application afin de pouvoir utiliser notre machine en tant que serveur. Premièrement, nous devions installer Apache afin de pouvoir utiliser notre machine comme serveur HTTP. Nous devions ensuite installer PostgreSQL, afin de pouvoir faire de la gestion de bases de données. Nous avons continué en installant PHP, afin de pouvoir mettre en ligne des pages comportant du code en PHP. Enfin, nous avons installé l'application PhpPgAdmin afin de pouvoir, après configuration, accéder à une version graphique de PostgreSQL en passant par le navigateur internet de la machine sur laquelle on se trouve. Pour finir, nous avons du mettre en ligne une page internet.
